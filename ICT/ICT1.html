Lesson 1

A. Browser Output

CSS

  • Stands for Cascading Style Sheets
  • Describes how html elements are to be displayed. Used to style html. It enhances design with layout, colors, and animations, making the webpage more attractive and pleasing. It also improves accessibility and responsiveness.
  • CSS saves a lot of work and saves time. It can control the layout of multiple web pages all at once. It makes maintenance easier.
  • Provides consistency across webpages.

Layers of a Webpage

  • Content Layer - define and layout the text, images, animation, sound, video and everything you want to put in a web page with the use of tags. Is the html code of the page. It serves as the foundation of the webpage.
  • Presention Layer - where you define how people see your webpage in a browser.
  • Behavior Layer - where you have a real-time user interaction with the web page. It deals with the interactions.

B. What I have Learned

In this lesson, I learned about Cascading Style Sheets, its uses, and its benefits. I learned that it is used to style webpages, and describe how the elements will appear on the user's screen, and in other medias. I found out that it is beneficial, as it separates design from content, making maintenance easier. It also lets us style multiple webpages at once, saving a lot of work. I also learned about the different layers of web programming, including the content layer, presentation layer, and behavior layer. Html belongs to the content layer, while CSS is in the presentation layer. I also learned what we need in each layer to make a good webpage.
