B. What I have Learned

In lesson 5, I learned about the different types of CSS selectors. Element selector selects the html elements. Id selector selects a specific html element with a unique identifier using a # followed by the id. Universal selector selects all the elements using *. Group selector selects multiple html elements separated by commas. Attribute selector selects html elements based on the value of a specific attribute. Lastly, class selector selects multiple elements that belong to the same class or category, and is denoted by . followed by the class name.
