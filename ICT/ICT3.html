B. What I have Learned

In this lesson, I learned about block and inline level elements. Block level elements include the div tag, paragraph tag, and many others. These are laid out vertically, taking up as much space as possible by default, including horizontal space. Each one of these starts on a new line. On the other hand, inline level elements display elements in a line, without forcing them to go to a new line. It includes anchor tag, and many others. You can change the display level of the element through the display property in CSS. I also learned about the div and span tags. Div tags divide html content into larger sections, while span tag is for inline and does not have line breaks. These do not have default styling information.
